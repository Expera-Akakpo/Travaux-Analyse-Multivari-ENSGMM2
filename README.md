Analyse de Données Multivariées (R & Python) 

Analyse en Composantes Principales (ACP) : Étude physico-chimique de cépages de vins (base Wine). Utilisation des bibliothèques FactoMineR (R) et Scikit-Learn (Python) pour la réduction de dimension et la visualisation (cercles de corrélation, projections d'individus).


Analyse Factorielle des Correspondances (AFC) : Analyse de tableaux de contingence sur la répartition des tâches ménagères (base HouseTasks) pour identifier les dépendances entre variables qualitatives.


Analyses Avancées : Mise en œuvre de l'Analyse des Correspondances Multiples (ACM) et de l'Analyse de Données Mixtes (AFMD).
